## Description

simple babel。旨在深入理解babel的词法分析，替换等核心功能。


目前支持严格模式'use strict',支持大部分词法。

Statement

- [x] VariableDeclaration
- [x] FunctionDeclaration
- [x] ExpressionStatement
- [x] BlockStatement
- [x] LabeledStatement
- [x] SwitchStatement
- [x] BreakStatement
- [x] ContinueStatement
- [x] EmptyStatement
- [x] ThrowStatement
- [x] ReturnStatement
- [x] IfStatement

Expression

- [x] thisExpression
- [x] SequenceExpression
- [x] AssignmentExpression
- [x] UpdateExpression
- [x] UnaryExpression
- [x] MemberExpression
- [x] LogicalExpression
- [x] BinaryExpression
- [x] CallExpression
- [x] FunctionExpression
- [x] VariableDeclarator
- [x] ObjectExpression
- [x] ArrayExpression
- [x] Property
- [x] Literal
- [x] Identifier
